📨 Telegram Bot แจ้งเตือน
ยังไม่ได้ตั้งค่า
วิธีตั้งค่า Telegram Bot:
1. เปิด Telegram → ค้นหา @BotFather → พิมพ์ /newbot → ได้ Bot Token
2. เพิ่ม Bot เข้า Group หรือแชทกับ Bot ตรงๆ
3. เปิด https://api.telegram.org/bot[TOKEN]/getUpdates → หา chat id
✅ ส่งตรงจาก browser ได้เลย ไม่มีปัญหา CORS

⚠️ ต้องสร้าง Table ใน Supabase ก่อน:
เปิด Supabase → SQL Editor → วาง SQL นี้แล้วกด Run:
CREATE TABLE IF NOT EXISTS bs_settings (id TEXT PRIMARY KEY, value TEXT, created_at TIMESTAMPTZ DEFAULT now());
ALTER TABLE bs_settings ENABLE ROW LEVEL SECURITY;
CREATE POLICY "public_read_write" ON bs_settings FOR ALL USING (true) WITH CHECK (true);
